# פרויקט מזכירה — לוז שנתי גמיש למכון רוטנברג

## מטרה
לבנות פלטפורמת יומן שתאפשר למזכירה האקדמית לנהל את הלוז של כל התוכניות במכון, לראות מה בכל תוכנית, מה התבטל, ומי מלמד. במקביל לשמש כמקור אמת לכל מי שצריך לראות (תלמידים/מדריכים/צוות — TBD).

## רקע
המזכירה ניסתה לבנות את זה ב-Lovable (`rotenberg-luz.lovable.app`). היא הצליחה להגדיר 8 תוכניות עם צבעים, חודשי, סינון ועוד — אבל בגרסה החיה הלוח החודשי ריק (118 מפגשים במונה, 0 מוצגים), והיא מסתבכת בתחזוקה.

## ההחלטה הארכיטקטונית — Google Calendar (אופציה B)
**תוקף:** מאושר ב-2026-03-04 בשיחה עם המשתמש.

**הסיבה:** Google Calendar עושה native את כל מה שהיא ניסתה לבנות:
- ריבוי יומנים עם צבע + checkbox לסינון
- תצוגת יום/שבוע/חודש/שנה
- חזרתיות (שיעור כל שני עד תאריך X = קליק אחד)
- ביטול שיעור = קליק אחד על אירוע יחיד או על סדרה
- מובייל + תזכורות + Hebrew calendar overlay
- חינם, בלי שרת, בלי תחזוקה
- ייצוא, שיתוף, embed
