# מסמך פרויקט — שיבוץ תלמידות לקבוצות

תת-פרויקט של `secretary-project/`. נוצר 2026-06-29. git: commit `f44281e`.
**סטטוס: ⏳ בנוי ומגובה — טרם נפרס (נועם יפרוס מאוחר יותר; אני אזכיר).**

## מטרה
כלי פשוט שמקבל רשימת שמות תלמידות, ואחת לכמה פגישות משבץ אותן לקבוצות קטנות של **2 או 4**, כך שעם הזמן כל אחת נפגשת עם כמה שיותר אחרות.

## החלטות יסוד (נועם, 2026-06-29)
| נושא | החלטה |
|---|---|
| שיבוץ | **חכם** — מגוון, מצמצם חזרות-זוגות, שומר היסטוריה |
| גודל קבוצה | **ניתן לבחירה בכל סבב — 2 או 4** (האחראים טרם החליטו; תומך בשתיהן) |
| ארכיטקטורה | **משותף** — Apps Script + Google Sheet (כמו שאר אפליקציות המכון) |
| מזהה DB | ב-**Script Properties** מההתחלה (`GS_DB_ID`) — עמיד בהדבקת config |

## איך זה עובד
1. **רשימת תלמידות** — שם בכל שורה → "שמירת רשימה" (מסיר כפילויות). נשמר בלשונית `תלמידות`.
2. **שיבוץ** — בוחרים זוגות/רביעיות → "שבץ" → רואים את הקבוצות → "ערבב שוב" (תוצאה אחרת) או "שמור סבב".
3. **שמירת סבב** — נשמר בלשונית `סבבים` עם תאריך/גודל/קבוצות.
4. **היסטוריה** — רשימת כל הסבבים; אפשר למחוק.

### האלגוריתם החכם (`generateGroups_` ב-lib.gs)
- בונה ספירת-זוגות (`pairCounts_`) מכל הסבבים השמורים: כמה פעמים כל שתי תלמידות היו יחד.
- גרידי: מערבב את הרשימה, ולכל קבוצה מצרף את התלמידה ה**פנויה עם הציון הנמוך ביותר** (סכום ספירת-הזוגות מול חברות הקבוצה הנוכחית) + שובר-שוויון אקראי.
- → עם הזמן הצירופים מתגוונים.
- **שארית:** אם נותרת תלמידה בודדת, היא מתמזגת לקבוצה הקודמת (קבוצה אחת בגודל שונה — 3/5).

## קבצים
`apps-script/`: `config.gs` (לשוניות + GS_DB_ID) · `setup.gs` (הקמת הגיליון) · `lib.gs` (נתונים + אלגוריתם) · `Code.gs` (Web App + API) · `index.html` (ממשק RTL). + `README.md` (שימוש/התקנה).

## פריסה (פרויקט Apps Script נפרד, תחת חשבון המכון)
1. פרויקט Apps Script **חדש**; הדבק את 4 קבצי ה-.gs + `index.html`.
2. הרץ **`setup()`** פעם אחת → נוצר הגיליון "שיבוץ תלמידות — App DB", id נשמר ב-Script Properties.
3. **Deploy → New deployment → Web app** (Execute as: Me · Access: לפי הצורך).
4. פתח את /exec → הזן שמות → שבץ.

## רעיונות להמשך (אופציונלי)
- "נעילת" זוגות שאסור/חובה יחד.
- ייצוא סבב (טקסט/וואטסאפ).
- הצגת "כמה פעמים כל אחת כבר נפגשה עם כל אחת" (מטריצת כיסוי).
